# rpg_project
Third person Action RPG prototype
Features:
1. Character:
   - Movement: WASD
   - Lock target: middle mouse
   - Attack: left mouse
   - Combo Attack: hold right-mouse for 0.5s then release
   - Damage
2. Enemy:
   - AI chasing character
   - Attack sequence: combo -> heavy
  

Developed with Unreal Engine 5
